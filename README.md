# Developer Portfolio

A professional portfolio website built with Laravel to showcase projects, skills, and testimonials.

## Features

- **Portfolio Website**: Showcase your projects, skills, and client testimonials
- **Admin Dashboard**: Easy management of content
- **Responsive Design**: Works on all devices
- **Image Upload**: Support for project images and avatars

## Quick Setup

### Requirements
- PHP 8.0+
- Composer
- MySQL
- XAMPP (recommended for Windows)
